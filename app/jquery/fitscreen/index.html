fitScreen.js

A jQuery plugin that resizes elements to fit the width and height of the browser view-port, which smoothly pulls the element up or down to show the whole content of the element when the user has stopped scrolling.

Download Download Minified

The html


 
<div class="fitscreen red"></div>
<div class="fitscreen blue"></div>
<div class="fitscreen yellow"></div>
<div class="fitscreen green"></div>

And the jQuery


// and the jQuery
<script>
      $('.fitscreen').fitScreen();
</script>
